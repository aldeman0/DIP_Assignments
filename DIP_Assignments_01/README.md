## Requirements

To install requirements:

```setup
python -m pip install -r requirements.txt
```


## Running

To run basic transformation, run:

```basic
python DIP_Assignment1-1.py
```

To run point guided transformation, run:

```point
python DIP_Assignment1-2.py
```
